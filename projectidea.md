# Webpage Controlled Pi SuperCollider Noise Machine

Have my pi connected to a speaker via aux, then have it host a webpage that I can go to (preferabbly on a tablet) and I can:
- Start/Stop audio
- Increase/Decrease Volume
- Change the active "noise"
    - populated from a local directory of .scd files on the pi

## Hardware
- Raspberry pi 3b
    - Arch Linux ARM
    - Ethernet
- extraneous aux speaker
    - starting with headphones
    - aux cable
- Client Device
    - Phone, Tablet, Laptop, Smart Fridge, something with internet and a browser
    - (on local network)
    - My purposes will end with a tablet but dev on laptop & pc
